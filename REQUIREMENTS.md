# CSV Editor Requirements

## Search Highlighting

- **Row/Cell toggle**: User can choose to highlight entire rows or only matching cells
- **Empty cell search (`""`)**: Matches empty cells; respects Row/Cell toggle

## Rainbow Columns

Two toggle buttons to color each column distinctly, making it easy to visually track data across rows:
- **Rainbow Background**: Colors each column's cell backgrounds
- **Rainbow Text**: Colors each column's text
