Create s simple stimulator to show the growth/depletion of an investment portfolio used to fund a person's retirement. Users should be able to adjust five key inputs using sliders:

1. Starting balance — your portfolio value at retirement
2. Annual withdrawal — how much you draw each year
3. Expected return — average annual investment return
4. Annual volatility — used for Monte Carlo simulation
5. Inflation rate — how fast your cost of living rises
6. Retirement duration — how many years to model

The chart should also include withdrawals.
